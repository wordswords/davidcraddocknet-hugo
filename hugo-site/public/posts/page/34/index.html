Bacula Scheduling

Bacula is a great open-source distributed backup program for Linux/UNIX systems. It is separated into three main components: One ‘Director’ - which sends messages to the other components and co-ordinates the backup One or more ‘File Demons’ - which ‘pull’ the data from the host they are installed from.
